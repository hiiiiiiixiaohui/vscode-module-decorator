### version 0.0.3

### 功能

- 查看当前文件的模块名称
- 单文件右键点击 `重新分析模块映射` 重新分析当前文件的模块名称
- 支持热模块更新配置信息（routes更新、pages更新）

### 配置字段

- `extensionmodulemap.moduleSourceMapPath`: 需要解析的模块路径，默认值为`./src/pages`
- `extensionmodulemap.routeConfigPath`: 需要解析的路由文件，默认值为`./config/routes.ts`
